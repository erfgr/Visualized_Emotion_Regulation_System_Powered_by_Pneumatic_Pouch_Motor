# Visualized_Emotion_Regulation_System_Powered_by_Pneumatic_Pouch_Motor
气动袋式电机驱动的可视化情绪调节系统
该项目是一个**“气动袋式电机（Pneumatic Pouch Motor）驱动的可视化情绪调节系统”**的作品型/研究型仓库，主要用于沉淀项目产出物（论文、展示材料、实验数据与图片/视频），而不是传统意义上的可运行代码工程。

## 1. 项目目标与核心概念
- **目标**：通过一种可交互、可感知的实体装置，把“情绪调节”过程进行**可视化呈现**，并借助**气动软体驱动**（袋式电机）输出动态形变/反馈，从而支持用户的情绪表达、引导或调节。
- **核心技术点**：以气动软体执行器（Pouch Motor）作为主要驱动方式，结合可视化交互呈现，形成“情绪—反馈—交互”的闭环体验。
